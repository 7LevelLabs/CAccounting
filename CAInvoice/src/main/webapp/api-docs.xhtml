API Overview

This API is being developed for integration Cloud Accounting with other information systems.

For example, you need to get a simple way to understand - when you need to stop providing some services to your client. Sign in Cloud Accounting, create the required artifacts, and use the persons/get/isok/id call.  If response is no "ok", you can calmly plug-they-off. :)

And you probably already noticed that calls to are simple, and integration with any information systems will be simple too.

Help with the API

Any questions on the use of the API we are pleased to accept on ca@7levellabs.com .
